# Simple react-three-fiber starter project

Includes:

- three.js
- react-three-fiber
- drei
- react-spring v9
- draco binaries
- Scene setup with OrbitControls and StandardEffects

## How to use

```
npx degit gsimone/r3f-starter my-project
cd my-project

yarn install
yarn start
```

Or just fork this repository 🤷‍♂️
